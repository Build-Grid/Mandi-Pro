# 🏠 MandiPro Wiki

Welcome to the **MandiPro** project wiki — the central knowledge base for developers, contributors, and operators.

MandiPro is a full-stack digital marketplace platform built with a Spring Boot backend and a React frontend, designed to manage firms, users, inventory, and trade operations.
